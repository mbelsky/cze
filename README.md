# Czech Verb Practice

A web application for practising Czech verb conjugation. Built with [Svelte](https://svelte.dev/) and [Vite](https://vite.dev/).

## Features

- Random verb exercises covering all four Czech conjugation groups (-á, -í, -uje, -e)
- Conjugation practice for all six pronoun forms (já, ty, on/ona/ono, my, vy, oni/ony/ona)
- Instant answer validation with correct answer feedback
- Clean, responsive UI with light/dark mode support

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
